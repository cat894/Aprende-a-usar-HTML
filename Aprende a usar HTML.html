¿Qué es HTML y para qué sirve?

HTML son las siglas de HyperText Markup Language, que en español significa Lenguaje de Marcado de Hipertexto. Pero no te asustes con el nombre, simplemente es un lenguaje especial que usamos para contarle a los navegadores cómo debe ser la estructura de una página web.

Puedes pensar en HTML como el esqueleto de un sitio web. Él organiza las cosas para que el navegador sepa dónde va el título, dónde va un párrafo de texto, dónde debe aparecer una imagen o un enlace para hacer clic.

Sin HTML, no habría páginas web, porque no sabríamos cómo poner los textos, imágenes ni botones en la pantalla.

La estructura básica de un archivo HTML

Un archivo HTML es un simple archivo de texto que termina con la extensión .html. Para verlo, solo necesitas abrirlo con un navegador (Chrome, Firefox, Edge, etc.).

Veamos cómo es el esqueleto básico de un archivo HTML:

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Mi primera página web</title>
  </head>
  <body>
    <h1>¡Hola mundo!</h1>
    <p>Estoy aprendiendo HTML.</p>
  </body>
</html>

¿Qué significa todo esto?

  • <!DOCTYPE html>: Esta línea le dice al navegador que usaremos HTML versión 5, que es la más moderna y estándar.
  • <html lang="es"> ... </html>: Esta etiqueta engloba toda la página web. El atributo lang="es" indica que el contenido está en español, para ayudar a buscadores y lectores de pantalla.
  • <head> ... </head>: Aquí dentro se pone información que no se ve directamente en la página, como el título que aparece en la pestaña del navegador, o los ajustes que ayudan a que la página funcione bien.
  • <meta charset="UTF-8">: Esto dice que la página usará una codificación universal para mostrar correctamente letras, acentos y símbolos.
  • <title> ... </title>: Es el título que aparece arriba, en la pestaña del navegador o cuando guardas la página en favoritos.
  • <body> ... </body>: Aquí va todo lo que sí quieres que se vea en la pantalla, como textos, imágenes, botones y más.
  • <h1> ... </h1>: Esta etiqueta crea un título grande y principal, normalmente el encabezado más importante de la página.
  • <p> ... </p>: Esto crea un párrafo de texto normal, como este que estás leyendo ahora.
Nota importante: Cada etiqueta HTML tiene una etiqueta de apertura (<nombre>) y una de cierre (</nombre>). Esto indica dónde empieza y termina esa parte del contenido. Hay algunas etiquetas que no necesitan cierre, como las imágenes (<img>).

Las etiquetas más importantes y para qué sirven

Cuando creas una página, usarás muchas etiquetas para diferentes cosas. Aquí te muestro las más usadas y qué hacen:

  • <h1> a <h6>: Son títulos de diferentes tamaños, desde el más grande (h1) hasta el más pequeño (h6).
  • <p>: Párrafo de texto.
  • <a href="URL"> ... </a>: Crea un enlace para ir a otra página o sitio cuando haces clic.
  • <img src="imagen.jpg" alt="Texto alternativo">: Muestra una imagen. El atributo alt es un texto que aparece si la imagen no carga y ayuda a personas con problemas visuales.
  • <ul> y <ol>: Listas, las ul son listas sin orden (con viñetas) y ol son listas ordenadas (numeradas).
  • <li>: Elementos dentro de listas.
  • <div>: Un bloque o contenedor para agrupar cosas y poder darles estilo o organizar mejor.

Ejemplo práctico con estas etiquetas

<body>
  <h1>Mis hobbies favoritos</h1>
  <p>Aquí te cuento qué me gusta hacer en mi tiempo libre.</p>

  <ul>
    <li>Leer libros de aventuras.</li>
    <li>Jugar videojuegos.</li>
    <li>Crear páginas web.</li>
  </ul>

  <p>Si quieres saber más, visita <a href="https://es.wikipedia.org">Wikipedia en español</a>.</p>

  <img src="https://via.placeholder.com/150" alt="Imagen ejemplo">
</body>

Este código crea un título, un párrafo, una lista con 3 elementos, un enlace y una imagen.

¿Cómo hacer tu primer archivo HTML y verlo?

  1. Abre un programa para escribir texto simple, como Notepad en Windows o TextEdit en Mac (configurado en modo texto plano).
  2. Copia y pega el código de ejemplo o el que hayas creado.
  3. Guárdalo con el nombre que quieras, pero con la extensión .html. Por ejemplo: mipagina.html
  4. Ubica el archivo guardado y haz doble clic para abrirlo en tu navegador favorito.

Consejos para que todo salga bien

  • Es importante cerrar todas las etiquetas que tienen apertura, para que el navegador entienda bien tu página.
  • Usa comentarios para dejar notas en tu código y recordar qué hace cada parte. Por ejemplo: <!-- Esto es un comentario -->.
  • Experimenta y prueba cambiar textos o agregar nuevas etiquetas para ver qué pasa.
  • Cuando quieras darle estilo y color, aprende sobre CSS, que es el lenguaje para decorar tus páginas.
  • Si quieres que tu página haga cosas interactivas (como botones que respondan), aprende JavaScript después de dominar HTML y CSS.
